## Reading Notes Sharing Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/14172ae6f3bf41c991fde781ea292f15/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Note Sharing
4. Communication Interaction
5. Announcement Information
6. Personal Center
7. Personal Note Management
8. Note Sharing Management


**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Note Sharing Management
5. Personal Note Management
6. Communication Interaction
7. Administrator Management
8. System Management
